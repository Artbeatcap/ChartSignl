# App Assets

Place the following images in this directory:

## Required Assets

### `icon.png`
- **Size:** 1024x1024 pixels
- **Purpose:** App store icon
- **Format:** PNG, no transparency

### `splash.png`
- **Size:** 1284x2778 pixels (iPhone 14 Pro Max)
- **Purpose:** Splash screen while app loads
- **Format:** PNG with transparency

### `adaptive-icon.png`
- **Size:** 1024x1024 pixels
- **Purpose:** Android adaptive icon foreground
- **Format:** PNG with transparency

### `favicon.png`
- **Size:** 48x48 pixels
- **Purpose:** Web browser favicon
- **Format:** PNG

## Design Guidelines

Keep assets consistent with ChartSignl's calm, habit-app aesthetic:

- **Colors:** Use teal/mint (#14B8A6) as primary
- **Style:** Simple, clean, minimal
- **Icon concept:** Consider a simple chart with key levels marked

## Generating Assets

You can use tools like:
- [Figma](https://figma.com) for design
- [Expo Icon Builder](https://buildicon.netlify.app/) for quick generation
- AI image generators for concepts

## Temporary Placeholders

For development, you can use solid color PNGs:
- Create 1024x1024 teal (#14B8A6) PNG for icon.png
- Create 1284x2778 light gray (#FAFAF9) PNG for splash.png
